Що таке хостинг?

Простими словами про складну технологію

Хостинг — це послуга з розміщення сайту в інтернеті. Уявіть, що ваш сайт — це книга. Щоб люди могли її прочитати, вона має лежати в бібліотеці. Хостинг — це та сама "бібліотека" для вашого сайту.

Сервер (потужний комп'ютер) працює цілодобово, зберігає файли сайту та віддає їх відвідувачам, коли ті вводять адресу у браузері.

Аналогія з життя

Хостинг = оренда квартири для сайту

• Квартира (сервер) має адресу (IP)
• Ви платите за оренду (хостинг-план)
• Чим більше квартира — тим більше гостей вміщається (відвідувачів)
• Хороший район = швидкий інтернет (швидкість завантаження)

Види хостингу

Різні потреби — різні рішення

🏠

Віртуальний хостинг

Один сервер ділиться між багатьма користувачами. Як багатоквартирний будинок — у кожного своя квартира, але спільні комунікації.

Ідеально для: блогів, візиток, невеликих магазинів

Хостинг-провайдери:
Hostinger, Bluehost, SiteGround

Сайти: невеликі блоги на WordPress, портфоліо
🏢

VPS (Віртуальний виділений сервер)

Сервер розділений на віртуальні машини. У вас є гарантована частина ресурсів. Як таунхаус — більше простору та контролю.

Ідеально для: інтернет-магазинів, додатків середнього розміру

Провайдери:
DigitalOcean, Linode (Akamai), Vultr

Сайти: середні e-commerce проєкти, SaaS-платформи
🏰

Виділений сервер

Весь фізичний сервер тільки ваш. Максимальна потужність та контроль. Як власний будинок — все під вашим керуванням.

Ідеально для: великих порталів, ігрових серверів, банків

Провайдери:
OVHcloud, Hetzner

Сайти: великі новинні портали, стримінгові сервіси, банківські системи
☁️

Хмарний хостинг

Ресурси беруться з мережі серверів. Автоматичне масштабування під навантаження. Платите лише за те, що використовуєте.

Ідеально для: стартапів, проєктів з непередбачуваним трафіком

Провайдери:
Amazon AWS, Google Cloud, Microsoft Azure

Сайти: Netflix, Airbnb, Uber — усі на хмарі
🔄

Реселлер хостинг

Ви купуєте ресурси оптом та перепродаєте їх своїм клієнтам. Як орендодавець, який здає квартири піднаймачам.

Ідеально для: веб-студій, агентств, розробників

Провайдери:
ResellerClub, HostGator Reseller

Сценарій: веб-студія створює сайти для клієнтів та продає їм хостинг під своїм брендом

Мови веб-програмування

Три кити сучасного вебу

HTML

Структура сторінки

HTML (HyperText Markup Language) — це кістяк сайту. Він визначає, де буде заголовок, де параграф, а де зображення.

<h1>Заголовок</h1>
<p>Абзац тексту</p>
JavaScript

Поведінка та інтерактив

JavaScript додає "життя" сторінці: реакції на кліки, анімації, перевірку форм, оновлення без перезавантаження.

<script>
  // Показати повідомлення
  alert('Привіт!');
</script>
PHP

Серверна логіка

PHP працює на сервері. Він генерує сторінки динамічно: підключається до бази даних, обробляє форми, створює особисті кабінети.

<?php
  $name = "Світ";
  echo "Привіт, $name!";
?>

Приклад: атрибут border

Атрибут border визначає товщину рамки навколо елемента. У сучасній розробці замість нього використовують CSS, але для розуміння структури це корисно:

<table border="2">
  <tr>
    <td>Клітинка 1</td>
    <td>Клітинка 2</td>
  </tr>
</table>

💡 Підказка: У сучасному CSS рамку задають через border: 2px solid black;
